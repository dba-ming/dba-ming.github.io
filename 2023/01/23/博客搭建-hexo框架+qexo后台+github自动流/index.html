Hexo介绍

Hexo 是一个基于 Node.js 的静态博客框架，它能够快速地生成静态网页，适合个人博客的搭建和管理。Hexo 的官方网站在开源社区中很早就开始受到关注，但具体何时开始火起来并不容易确定。Hexo 的开发始于2012年，最初是由台湾的一位开发者 Tommadness（也称为 Tom Chen）发起的，其代码在 GitHub 上进行了开源。随着时间的推移，Hexo 的功能不断完善，社区也逐渐壮大，吸引了越来越多的用户和贡献者。可以说，Hexo 在过去几年里逐渐走红，成为了静态网站生成器领域的一大翘楚。

Hexo 官网：https://hexo.io/zh-cn/

Hexo 官方文档：https://hexo.io/zh-cn/docs/

Hexo Fluid 主题用户手册 ：https://hexo.fluid-dev.com/docs/guide

Hexo Github：https://github.com/hexojs/hexo

qexo介绍

Qexo 是一个快速、强大、美观的在线静态博客编辑器。使用 GPL3.0 开源协议。支持包括且不限于在 Vercel 等平台部署, 为您的静态博客添加动态的元素。

特点

  • Qexo拥有直观的用户界面和易于使用的编辑器，使得用户可以方便地创建和管理他们的博客内容；
  • 支持多种图床上传, 上传尽在弹指之间；
  • 支持将文章、页面、配置索引一键缓存至数据库, 保证您的高速访问；
  • Webhook 全自动清除缓存, 时刻保持数据最新；
  • 模块化架构；
  • 支持多种 Hexo、Hugo、Valaxy 托管商 Github、Gitlab、本地；
  • 多种图床协议支持 Github、S3、FTP、远程API；
  • Markdown 语法 + 多种编辑界面, 所见即所得；
  • 通知中心 通知栏 + 红点标记；
  • 多形式推送 Bark、Telegram、Pushdeer、Wechat；
  • reCaptcha 防止垃圾信息侵害；
  • 友情链接 前端申请, 一键接入；
  • 自动更新 简单方便, 保持最新；
  • 自定字段 / 站点统计 / 页面管理 / 配置编辑；
  • 评论通知 / 图片上传 / 标识生成 / API 拓展。

Github：https://github.com/Qexo/Qexo/

github自动流

GItHub Actions是一个持续集成和持续交付的平台，能够让你自动化你的编译、测试和部署流程。

如下，是qexo编辑器触发的一次github自定义工作流：

  • 检出仓库 # Checkout
  • 安装 Node.js
  • 安装 Hexo
  • 缓存 Node 插件 # Cache Modules
  • 安装依赖
  • 生成静态文件 # Generate
  • 部署到 github page
  • 推送到服务器私有仓库
  • 刷新CDN
